# Lets Warm Up

## Challenge

The challenge asks what ASCII character corresponds to the hexadecimal value `0x70`.

---

## My Approach

1. Recognized that `0x70` is a hexadecimal value.
2. Converted the hexadecimal value to its ASCII equivalent.
3. Found that `0x70` represents the character **`p`**, which is the beginning of the flag.

---

## Commands/Tools Used

- ASCII Table
- Hex to ASCII Converter

---

## What I Learned

- How hexadecimal values map to ASCII characters.
- That every printable character has a unique ASCII value.
- How to convert between hexadecimal and ASCII.

---

## Key Takeaway

Understanding ASCII and hexadecimal representations is a fundamental skill for CTF challenges involving encoding, binary analysis, and reverse engineering.
